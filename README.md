# AI-Agent

> **Setup:**
> ```bash
> python3.11 -m venv .venv && source .venv/bin/activate
> pip install -r requirements.txt
> git clone https://github.com/bazinga012/mcp_code_executor.git
> cd mcp_code_executor
> npm install
> cd ..
> ```
